# Visión del Proyecto (Project Vision)

## 1. Declaración de la Visión
Desarrollar una aplicación de comercio electrónico (SPA) moderna, escalable y mantenible, que sirva como referencia arquitectónica para proyectos React profesionales. El objetivo es demostrar la implementación efectiva de **Feature-Sliced Design (FSD)**, principios **SOLID** y una experiencia de usuario (UX) pulida, utilizando un stack tecnológico minimalista pero robusto.

## 2. Objetivos de Negocio
*   **Demostración Técnica:** Servir como portfolio avanzado para demostrar capacidades de Arquitectura de Software Frontend.
*   **Escalabilidad:** Proveer una base de código que permita agregar nuevas funcionalidades (ej. autenticación, pasarela de pago) sin refactorizaciones masivas.
*   **Performance:** Asegurar tiempos de carga rápidos y una interfaz reactiva mediante gestión de estado optimizada.

## 3. Alcance (In-Scope / Out-Scope)

### ✅ In-Scope (Dentro del Alcance)
*   Visualización de catálogo de productos (desde API externa).
*   Gestión de carrito de compras (agregar, eliminar, persistencia en sesión).
*   Cálculo de totales y resumen de orden.
*   Diseño totalmente responsivo (Mobile First).
*   Arquitectura modular estricta.

### ❌ Out-Scope (Fuera del Alcance)
*   Backend propio (se usa FakeStoreAPI).
*   Procesamiento real de pagos.
*   Autenticación de usuarios (Login/Registro) - *Planeado para Fase 2*.
*   SSR (Server Side Rendering).
